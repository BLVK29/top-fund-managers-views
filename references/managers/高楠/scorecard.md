# 高楠框架评分卡

> 用高楠的投资方法给任意基金打分，衡量"与高楠投资风格的契合度"。

## 六维评分

| 维度 | 权重 | 评分依据 |
|---|---|---|
| 1. 胜率型资产含量 | 25分 | 重仓股是否有业绩高增长确定性？未来2-3年能否快速放量？ |
| 2. 赔率型资产含量 | 20分 | 是否有估值低位、基本面见底、有催化剂的标的？赔率空间如何？ |
| 3. 积木式组合 | 15分 | 行业是否多元分散？板块间相关性是否低？前十大集中度是否较低（<50%）？ |
| 4. 前瞻性布局 | 15分 | 是否有左侧/前瞻性持仓？是否在市场尚未关注时介入？ |
| 5. 动态调整与止盈 | 15分 | 换手率是否合理？是否有止盈动作？是否动态优化A/H股估值差异？ |
| 6. 业绩印证 | 10分 | 任职回报如何？是否打造过翻倍基？回撤控制如何？ |

## 评级标准

| 总分 | 评级 | 含义 |
|---|---|---|
| 80-100 | 高度契合 | 非常像高楠会买的基金 |
| 60-79 | 较为契合 | 有部分高楠风格特征 |
| 40-59 | 一般 | 风格中性，部分维度契合 |
| 20-39 | 不太契合 | 风格偏离较大 |
| 0-19 | 不契合 | 风格几乎相反 |

## 打分流程

1. 运行 `python scripts/score_fund.py <代码或名称> --manager 高楠`
2. 脚本自动备好数据
3. 按上表六维逐项给分
4. 给总分、评级、理由，并和高楠自己的基金对比风格

## 重要提醒

- 这套分衡量的是"**与高楠投资风格的契合度**"，不是基金好坏的绝对评判
- 高集中度/低换手/纯价值型基金天然低分很正常
- 个股业绩确定性、产业趋势判断等无数据项标"需核实"
