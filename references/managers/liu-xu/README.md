# Liu Xu Data Collection

This folder contains the Stage 1 data collection package for fund manager Liu Xu.

## Structure

### profile.md

Fund manager profile.

### method.md

Draft investment methodology extracted from reports.

### corpus/

Collected corpus from:

- Annual reports
- Interim reports
- Quarterly reports
- Media interviews
- Public webpages

### fund_data/

Structured data including:

- representative funds
- report index
- holdings snapshot
- holding overlap analysis

### sources.csv

Complete source inventory.
