# 谢治宇框架评分卡

> 用谢治宇的投资方法给任意基金打分，衡量"与谢治宇投资风格的契合度"。

## 六维评分

| 维度 | 权重 | 评分依据 |
|---|---|---|
| 1. 均衡度 | 25分 | 行业是否分散？是否押注单一赛道？前十大集中度是否中等（50-60%）？ |
| 2. 自下而上选股 | 20分 | 重仓股是否为具有长期成长性的优质企业？有无差异化竞争力？ |
| 3. 大类配置视野 | 15分 | 是否有港股配置？是否关注股债关系/黄金等逆周期资产？ |
| 4. 估值合理性 | 15分 | 重仓股估值是否合理？是否在低估值时逆向布局？ |
| 5. 时代脉搏 | 15分 | 是否关注科技创新（半导体/AI）？是否关注高分红回报？两大定价力量？ |
| 6. 业绩印证 | 10分 | 长期业绩是否跑赢基准？任职回报如何？波动控制如何？ |

## 评级标准

| 总分 | 评级 | 含义 |
|---|---|---|
| 80-100 | 高度契合 | 非常像谢治宇会买的基金 |
| 60-79 | 较为契合 | 有部分谢治宇风格特征 |
| 40-59 | 一般 | 风格中性，部分维度契合 |
| 20-39 | 不太契合 | 风格偏离较大 |
| 0-19 | 不契合 | 风格几乎相反 |

## 打分流程

1. 运行 `python scripts/score_fund.py <代码或名称> --manager 谢治宇`
2. 脚本自动备好数据
3. 按上表六维逐项给分
4. 给总分、评级、理由，并和谢治宇自己的基金对比风格

## 重要提醒

- 这套分衡量的是"**与谢治宇投资风格的契合度**"，不是基金好坏的绝对评判
- 极度集中型/单一行业基金天然低分很正常
- 个股基本面数据无数据项标"需核实"
