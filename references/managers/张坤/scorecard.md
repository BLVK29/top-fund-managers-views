# 张坤框架评分卡

> 用张坤的投资方法给任意基金打分，衡量"与张坤投资风格的契合度"。

## 六维评分

| 维度 | 权重 | 评分依据 |
|---|---|---|
| 1. 生意模式与竞争壁垒 | 25分 | 前十大重仓是否为生意模式好、有壁垒的公司？自由现金流是否充沛？ |
| 2. 长期成长性 | 20分 | 重仓股是否有长期持续成长性？成长是否高质量（合理投资回报率下获得）？ |
| 3. 管理层与治理 | 15分 | 管理层是否审慎配置资本？有无分红/回购？有无乱投资？ |
| 4. 估值合理性 | 15分 | 当前估值相对历史/全球是否合理？自由现金流回报率如何？估值溢价是否收敛？ |
| 5. 持仓集中度与换手 | 15分 | 集中度是否较高（张坤偏好集中）？换手率是否低（长持）？ |
| 6. 业绩印证 | 10分 | 长期业绩是否跑赢基准？回撤控制如何？任职回报如何？ |

## 评级标准

| 总分 | 评级 | 含义 |
|---|---|---|
| 80-100 | 高度契合 | 非常像张坤会买的基金 |
| 60-79 | 较为契合 | 有部分张坤风格特征 |
| 40-59 | 一般 | 风格中性，部分维度契合 |
| 20-39 | 不太契合 | 风格偏离较大 |
| 0-19 | 不契合 | 风格几乎相反 |

## 打分流程

1. 运行 `python scripts/score_fund.py <代码或名称> --manager 张坤`
2. 脚本自动备好数据（张坤的用精编快照，别的实时抓取并缓存）
3. 按上表六维逐项给分
4. 给总分、评级、理由，并和张坤自己的基金对比风格

## 重要提醒

- 这套分衡量的是"**与张坤投资风格的契合度**"，不是基金好坏的绝对评判
- 防御型/红利/纯债基金天然低分很正常，要讲清楚，别让用户误以为"低分=差基金"
- 个股ROE、自由现金流等无数据项标"需核实"
