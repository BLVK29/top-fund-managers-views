# WorkBuddy 部署说明

## 1. 安装

将整个 `top-fund-managers-views` 文件夹复制到 WorkBuddy 的技能目录：

```
~/.workbuddy/skills/top-fund-managers-views/
```

## 2. 导入

1. 打开 WorkBuddy 设置
2. 进入"技能"页面
3. 点击"导入技能"
4. 选择 `skill.yml` 文件
5. 启用技能

## 3. 依赖安装

```bash
pip install -r requirements.txt
```

仅需 `requests`、`beautifulsoup4`、`lxml` 三个库。

## 4. 功能验证

部署后可用以下方式验证：

### 语料检索
```
问：张坤怎么看白酒行业？
问：谢治宇的投资方法是什么？
问：高楠怎么看AI算力？
```

### 基金数据
```
问：张坤的易方达蓝筹精选持仓了什么？
问：谢治宇的兴全合润业绩如何？
问：高楠的永赢睿信规模多大？
```

### 三经理对比
```
问：张坤和高楠的投资方法有什么本质区别？
问：三位基金经理谁更擅长控制回撤？
```

### 评分
```
问：用张坤的标准给招商中证白酒打个分
```

## 5. 注意事项

- 语料检索不需要联网
- 全市场基金抓取需要联网（天天基金网）
- 持仓数据为季度快照，非实时
